# Rust in Godot ~4.3   
 
Just a cheatsheet to achieve the goal.  
 
```
Game/
├── assets/
│   ├── scripts/
│   │   └── rust/
│   │       ├── src
│   │       ├── .gdignore
│   │       ├── Cargo.lock
│   │       └── Cargo.toml
├── rust.gdextension
├── ..
```

> [!IMPORTANT]
